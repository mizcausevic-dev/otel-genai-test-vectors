# Security Policy

`otel-genai-test-vectors` ships static JSON conformance fixtures and a thin Node loader. No network listener, no remote fetch, no execution of user-supplied code, no secrets.

All vector content is **synthetic**. No real prompt content, completion text, customer identifiers, or production identifiers are included.

## Supported versions

Only the latest tagged release is supported.

## Reporting a vulnerability

Please use GitHub Security Advisories for private disclosure:

- [Open a security advisory](https://github.com/mizcausevic-dev/otel-genai-test-vectors/security/advisories/new)

Do not file public issues for security reports.
